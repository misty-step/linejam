PASS-THE-POEM PARTY GAME

Write one line. See only the line before yours. Read the chaos aloud together.

Linejam is a party game for friends in the same room. Nine rounds, word counts 1-2-3-4-5-4-3-2-1, and a reveal ceremony where everyone finally sees the whole poem they helped write without ever seeing it coming. The constraint is the game: no single person could have written what you end up with.

Wedding-validated - real friends, real reveal night, real laughter.
