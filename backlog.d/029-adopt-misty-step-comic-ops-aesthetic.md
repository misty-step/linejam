# Adopt the Misty Step comic-ops aesthetic baseline

Priority: P2 · Status: pending · Estimate: M

## Goal

Evaluate and adopt the romance-zine comic-ops flavor for Linejam's room,
writing, reveal, and recap surfaces.

## Oracle

- [ ] `DESIGN.md` or project docs name the chosen flavor, likely
      `romance-zine`, and the mobile-first surfaces it governs.
- [ ] One representative room/write/reveal/recap flow is rendered or mocked
      with comic-ops tokens while preserving fast phone play.
- [ ] Aesthetic changes do not add onboarding friction, extra game modes, or
      decorative complexity that dilutes the word-count constraint.
- [ ] The implementation uses `@misty-step/aesthetic` commit `9bbe0f9` or later,
      or records a deliberate no-adoption decision.
- [ ] `pnpm ci:prepush` or the current canonical gate passes after
      implementation.
